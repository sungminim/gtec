* 테이블 셀 기본은 중앙정렬.

  전체 좌측정렬일때는 table태그에 tdL 클래스 추가. 예) <table class="lineTop_tb tdL">
  부분 좌측정렬일때는 td태그에 txtL 클래스 추가. 예) <td class="txtL">학기 개강전</td>
* lineTop_tb는 항목이 위에 있는 표 / lineTop_tbL는 항목이 좌측에 있는 표.
* lineTop_tb, lineTop_tbL는 all line 표 / lineTop_tb2, lineTop_tbL2 좌우 line 없는 표.

[컨텐츠] 검색옵션

검색옵션 규칙
* .optUnit은 옵션 줄간 영역 / .labelUnit은 옵션 항목
<div class=“optUnit opt3”> : 한 줄 옵션에 항목이 3개 있을때
<div class=“optUnit opt2”> : 한 줄 옵션에 항목이 2개 있을때
<div class=“optUnit opt1”> : 한 줄 옵션에 항목이 1개 있을때
* .labelUnit에 있는 .sort2~.sort10는 항목글자수
<div class=“labelUnit에 sort2”> : 항목글자수가 2글자인 옵션
<div class=“labelUnit에 sort10”> : 항목글자수가 10글자인 옵션
* .optUnit이나 .labelUnit에 추가 class가 없으면 기본셋팅값

* .BBS_boxOption .opt_각네이밍의 추가 class가 있으면 별도 셋팅된 옵션
ex) <div class=“BBS_boxOption opt_subjct_name”> : 과목선택+성명(학번) 옵션
